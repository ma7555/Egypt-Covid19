COVID-19 Egyptian Monitoring

This project is open initiative to provide a minimal monitoring platform for the Egyptian covid19 situation.
Start by clicking one of the available links in the nav bar:

  • home for this home page
  • Cumulative Data for all the data gathered from the breakout
  • Daily Data for day by day updates on the situation
  • Statistics for useful statistics on the data
